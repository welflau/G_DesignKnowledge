# Hades 知识库

《Hades》游戏策划知识库，包含玩法、系统和数值三个维度的设计知识。

## 文件说明

| 文件名 | 内容 | 说明 |
|--------|------|------|
| Hades_玩法知识库.csv | 玩法设计 | 战斗操作、武器设计、Boss战、Build构建等 |
| Hades_系统知识库.csv | 系统设计 | 整体架构、成长系统、祝福系统、叙事系统等 |
| Hades_数值知识库.csv | 数值设计 | 伤害公式、属性数值、资源经济、难度曲线等 |

## 数据格式

CSV 格式，包含两列：
- **知识名**: 知识条目的标题
- **知识内容**: 详细的知识描述
