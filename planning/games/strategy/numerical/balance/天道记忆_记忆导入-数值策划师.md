```
暴击率 = 基础暴击 + 装备加成
```

## 四、设计原则

### 4.1 平衡性原则
1. **没有无敌套路**：任何单一属性堆叠都有边际效应
2. **风险与收益成正比**：高收益活动伴随高风险
3. **追赶机制**：落后玩家有加速成长途径
4. **软上限**：数值增长有天花板，防止无限膨胀

### 4.2 修仙感原则
1. **越往后越难**：符合修仙小说设定
2. **机缘重要性**：随机事件收益可能超过苦修
3. **资源竞争**：优质资源有限，促进玩家互动
4. **境界压制**：高境界对低境界有明显优势，但非绝对
