# 模板：MOBA/竞技游戏英雄故事

> 适用品类：MOBA/英雄射击/竞技
> 参考游戏：英雄联盟、王者荣耀、守望先锋

## 英雄叙事结构

### 基础设定
| 字段 | 说明 |
|------|------|
| hero_name | 英雄名 |
| hero_title | 称号 |
| faction | 阵营/地区 |
| role | 定位（战士/法师/刺客/辅助等） |
| one_liner | 一句话概括（15字内） |
| personality_tags | 性格标签（最多3个） |

### 叙事层级
| 层级 | 内容 | 字数 | 展示位置 |
|------|------|------|---------|
| L1-标签 | 称号+一句话 | 15字 | 选人界面 |
| L2-简介 | 身份+核心冲突 | 100字 | 英雄详情页 |
| L3-传记 | 完整背景故事 | 500-1000字 | 英雄故事页 |
| L4-宇宙 | 短篇故事/漫画 | 2000-5000字 | 官网/漫画 |
| L5-交互 | 英雄间对话/彩蛋 | 单条10-30字 | 游戏内 |

### 英雄间关系网
| 关系类型 | 说明 | 触发机制 |
|---------|------|---------|
| 宿敌 | 核心对立关系 | 击杀专属语音 |
| 战友 | 共同阵营 | 同队开场语音 |
| 师徒 | 传承关系 | 互动彩蛋 |
| 暧昧/CP | 粉丝向关系暗示 | 选人阶段互动 |

### 叙事设计要点
1. **服务于竞技辨识度**：叙事的首要目标是让英雄有辨识度和记忆点
2. **轻量但有深度**：核心玩家不看故事，但看了的玩家要能"上头"
3. **关系网驱动二创**：英雄间的化学反应是社区内容生产的核心动力
4. **文化适配**：王者荣耀用中国历史人物重塑，英雄联盟构建架空宇宙
5. **持续更新**：通过赛季/版本更新推进世界观（如英雄联盟的虚空事件）
