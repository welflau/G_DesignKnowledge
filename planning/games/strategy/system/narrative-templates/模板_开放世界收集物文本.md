# 模板：开放世界收集物/环境叙事文本

> 适用品类：开放世界/ARPG
> 参考游戏：原神（书籍/遗迹石板）、艾尔登法环（物品描述）、塞尔达

## 收集物类型

### 1. 书籍/手记
| 字段 | 说明 |
|------|------|
| book_id | 唯一ID |
| title | 书名 |
| author | 作者（可匿名） |
| volume | 卷数（系列书籍） |
| location | 获取位置描述 |
| content | 正文（500-2000字/卷） |
| worldlore_tag | 关联的世界观标签 |

### 2. 物品描述（风味文本）
| 字段 | 说明 |
|------|------|
| item_id | 唯一ID |
| item_name | 物品名 |
| flavor_text | 风味文本（50-150字） |
| lore_connection | 关联的世界观事件 |

### 3. 环境叙事元素
| 字段 | 说明 |
|------|------|
| env_id | 唯一ID |
| type | 类型（石碑/壁画/残骸/日记/信件） |
| text | 文本内容 |
| visual_desc | 视觉描述（给关卡美术） |
| context_clue | 玩家应从中获取的信息 |

### 叙事设计要点
1. **碎片化但有体系**：单个收集物自成一体，多个组合揭示完整故事
2. **奖励好奇心**：在偏僻角落放置重要世界观线索
3. **不同视角**：同一事件用不同作者/不同时间的文本描述
4. **保留悬念**：永远不要在收集物中给出100%的答案
5. **风味文本要有角色感**：《艾尔登法环》的物品描述是其叙事的核心载体
