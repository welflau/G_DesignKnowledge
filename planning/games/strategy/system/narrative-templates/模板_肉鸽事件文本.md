# 模板：肉鸽游戏事件文本

> 适用品类：肉鸽/Roguelike/Roguelite
> 参考游戏：杀戮尖塔、哈迪斯、以撒的结合

## 事件结构

### 基础字段
| 字段 | 说明 | 示例 |
|------|------|------|
| event_id | 事件唯一ID | evt_merchant_001 |
| event_name | 事件名称 | 神秘商人 |
| event_type | 类型 | encounter/combat/treasure/shop/rest/boss |
| floor_range | 出现层数范围 | [1, 15] |
| weight | 出现权重 | 100 |

### 叙事字段
| 字段 | 说明 | 示例 |
|------|------|------|
| description | 事件描述（进入时展示） | "你遇到了一个神秘的旅行商人，他的背包里似乎装满了奇怪的东西……" |
| options[] | 选项列表 | |
| options[].text | 选项文本 | "查看商品" / "离开" / "[力量>10] 威胁他" |
| options[].condition | 触发条件 | {"stat": "strength", "op": ">=", "value": 10} |
| options[].result_text | 选择后的叙述 | "商人被你的气势震慑，颤抖着打开了背包……" |
| options[].rewards | 奖励/惩罚 | {"gold": -50, "item": "random_rare"} |

### 叙事设计要点
1. **简短有力**：每段文字不超过3句话，玩家在高频决策中没有耐心读长文
2. **选项要有意义**：每个选项对应不同的风险/收益权衡
3. **角色一致性**：NPC的语言风格要在整个run中保持一致
4. **回合叙事**：利用死亡循环推进meta剧情（参考哈迪斯）
5. **条件文本**：根据玩家当前状态/历史选择展示不同文本
