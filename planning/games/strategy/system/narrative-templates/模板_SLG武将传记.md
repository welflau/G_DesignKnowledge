# 模板：SLG武将/英雄传记

> 适用品类：SLG/策略/4X
> 参考游戏：三国志系列、文明系列、万国觉醒

## 传记结构

### 基础信息
| 字段 | 说明 | 示例 |
|------|------|------|
| hero_id | 武将ID | hero_cao_cao_001 |
| name | 姓名 | 曹操 |
| title | 称号 | 乱世枭雄 |
| era | 所属时代 | 东汉末年 |
| faction | 阵营 | 魏 |
| rarity | 稀有度 | SSR |
| portrait_desc | 立绘描述（给美术的参考文字） | "身着暗金色铠甲，头戴束发金冠……" |

### 传记文本（分段解锁）
| 解锁条件 | 传记内容 |
|----------|---------|
| 获得时 | 简短介绍（50字内）：身份+最核心标签 |
| 突破1 | 早年经历（200字）：出身背景+性格形成 |
| 突破2 | 崛起之路（200字）：关键转折事件 |
| 突破3 | 巅峰时刻（200字）：最辉煌的成就 |
| 突破4 | 晚年/结局（200字）：命运归宿 |
| 满级 | 秘闻/彩蛋（100字）：不为人知的一面 |

### 台词设计
| 场景 | 台词 | 要求 |
|------|------|------|
| 出战 | "宁教我负天下人！" | 体现核心性格 |
| 胜利 | — | 霸气/从容 |
| 失败 | — | 不服/反思 |
| 待机闲聊 | — | 展现另一面 |
| 获得时 | — | 自我介绍式 |

### 叙事设计要点
1. **历史与游戏的平衡**：基于史实但为游戏性做适度改编
2. **性格极端化**：每个武将有一个核心性格标签（枭雄/忠义/智慧/勇猛）
3. **阵营叙事一致性**：同阵营武将的传记应互相呼应
4. **文字风格统一**：决定是文言/半文言/白话，全项目统一
