# SubSkill: 数值拆解

## 触发条件
用户需要拆解某款游戏的数值体系

## 工作流程
1. 确认目标游戏 → 定位 numerical/game-data/{game}/ 目录
2. 加载核心配表（角色表/技能表/敌人表）
3. 分析属性分布、成长曲线、伤害公式
4. 输出拆解报告

## 输出模板
```markdown
## [游戏名] 数值拆解报告

### 1. 属性体系
[属性列表/主属性/副属性/成长规则]

### 2. 伤害公式还原
[公式/验证数据]

### 3. 成长曲线
[等级-属性曲线/养成节奏]

### 4. 经济系统
[货币/产出/消耗/深度]

### 5. SLG策划启示
[哪些设计可以借鉴]
```
