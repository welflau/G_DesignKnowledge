# SubSkill: GDD文档生成

## 触发条件
用户需要生成GDD、策划案、系统文档

## 优先检索条目
1. `system/gdd-templates/GDD模板_3_Game Overview.md` — 游戏概览
2. `system/gdd-templates/GDD模板_4_Gameplay and Mechanics.md` — 玩法机制（核心）
3. `system/gdd-templates/GDD模板_5_Story, Setting and Character.md` — 世界观
4. `system/gdd-templates/GDD模板_15_Game Type Adaptation.md` — 类型适配

## 工作流程
1. 确认游戏品类 → 加载对应品类适配模块
2. 按GDD模板顺序逐章生成
3. 第3章和第4章为核心，必须详细
4. 输出为完整可交付的策划文档
