## Game Narrative Design: Complete Storytelling Guide

### 一、三大核心支柱
1. **剧情结构** — 激励事件→上升动作→高潮→结局，需适应玩家自由探索
2. **角色驱动叙事** — 角色是情感锚点，需有清晰动机和有意义转变
3. **世界构建基础** — 一致的规则、历史、文化和冲突

### 二、玩家选择类型
| 类型 | 说明 | 适用场景 |
|------|------|---------|
| 二元选择 | 是/否决定 | 教程、关键节点 |
| 道德困境 | 无明确正确答案 | 增强情感投入 |
| 资源管理 | 战略分配选择 | SLG/策略类 |
| 关系决策 | 影响角色关系和联盟 | RPG/SLG |
| 时限选择 | 带时间压力 | 增加紧迫感 |

### 三、五种分支叙事架构
1. **线性+侧支** — 主线一致，选择创建临时分支后汇合。如《巫师3》
2. **关键路径分支** — 重大决定产生永久分支。如《质量效应》
3. **网状结构** — 多条叙事线互相关联。如《底特律：变人》
4. **平行叙事** — 不同路径偶尔交叉。如《火焰纹章：风花雪月》
5. **中心-辐射** — 中央枢纽+分支内容。常见开放世界RPG

### 四、角色弧线类型
- **正向变化弧** — 克服缺陷并成长
- **负向变化弧** — 缺陷导致堕落
- **平坦弧** — 维持原则，改变世界（如猎魔人杰洛特）
- **玩家决定弧** — 反映玩家道德选择（如谢帕德）
- **关系驱动弧** — 通过与玩家互动而改变

### 五、世界构建七维度
1. 地理与气候
2. 历史与时间线
3. 文化与社会
4. 政治结构
5. 经济与贸易
6. 魔法/科技系统
7. 宗教与哲学

### 八、不同品类叙事要点
| 品类 | 叙事要点 |
|------|---------|
| **RPG** | 最深度投入，数百小时内容，分支+同伴关系 |
| **动作冒险** | 叙事配合动作节奏，如《神秘海域》 |
| **解谜** | 不干扰解谜，环境叙事为主，如《传送门》 |
| **恐怖** | 氛围优先，模糊性胜于显式叙述 |
| **SLG/在线服务** | 赛季叙事+世界事件，如《命运2》 |

### 九、任务设计原则
- 清晰的目标指引
- 每个任务都有故事背景
- 混合战斗/探索/解谜/社交
- 有意义的选择和可见后果
