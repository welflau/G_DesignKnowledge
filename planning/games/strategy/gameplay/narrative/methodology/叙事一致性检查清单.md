## 正文
## 叙事一致性检查清单

### 一、世界观一致性
- [ ] 物理法则是否贯穿所有内容（魔法规则/科技水平）
- [ ] 各地区的文化/建筑/服饰是否自洽
- [ ] 经济描述是否与实际游戏经济匹配
- [ ] 阵营/势力的实力对比是否与叙事描述一致
- [ ] 宗教/信仰体系是否前后统一
- [ ] 距离/时间描述是否与地图实际匹配

### 二、角色一致性
- [ ] 每个角色的行为是否符合其性格设定
- [ ] 角色动机是否有清晰的驱动逻辑
- [ ] 角色之间的关系在不同剧情中是否一致
- [ ] 角色的能力表现是否与设定匹配（不能叙事中是废柴但数值是最强）
- [ ] 角色语言风格是否贯穿（不能一会文雅一会粗鄙）
- [ ] 同一角色在不同任务中的称呼是否统一

### 三、时间线一致性
- [ ] 事件发生顺序是否合理
- [ ] 角色年龄与事件时间是否匹配
- [ ] 支线任务的时间点是否与主线矛盾
- [ ] NPC对话是否反映了当前剧情进度（已完成的事件NPC还在说"快去做"）
- [ ] 赛季/版本更新后旧内容是否需要调整

### 四、命名一致性
- [ ] 人名在所有出现处拼写一致
- [ ] 地名在所有出现处拼写一致
- [ ] 专有名词（技能名/道具名/阵营名）全局统一
- [ ] 同一概念是否用了不同词汇表述

### 五、逻辑一致性
- [ ] 玩家选择是否被后续内容正确响应
- [ ] 分支汇合后是否有不应出现的角色/信息
- [ ] 任务前置条件是否合理（不会出现"先遇到A才能完成B"但B先于A开放）
- [ ] 死亡/失败的角色是否在后续错误出现
- [ ] 隐藏信息的揭示时机是否合理

### 六、叙事-机制一致性（避免Ludonarrative Dissonance）
- [ ] 紧急剧情中玩家是否仍可自由闲逛
- [ ] "弱小"角色的数值是否确实弱小
- [ ] "同伴关系"是否在玩法中有体现
- [ ] 叙事中的"稀缺资源"在游戏中是否真的稀缺
- [ ] 战斗剧情中的"苦战"在玩法中是否有相应难度


## 策划参考价值
叙事QA的核心工具。每次提交剧情文档前过一遍此清单。
