## 极乐迪斯科叙事设计分析

### 核心创新：技能即内心声音

极乐迪斯科最革命性的设计——24个"技能"不是战斗能力，而是主角脑海中的**不同人格面**：

| 属性 | 技能 | 内心声音 | 对话风格 |
|------|------|---------|---------|
| 智力 | 百科全书 | 你内心的Wikipedia | 大段学术考据 |
| 智力 | 修辞学 | 你的话术专家 | 识破/使用修辞陷阱 |
| 精神 | 内陆帝国 | 你的直觉与想象力 | 诡异的联想和预感 |
| 精神 | 浮光掠影 | 你的品味 | 对美学/穿着的评价 |
| 体质 | 电化学 | 你的成瘾面 | 怂恿你喝酒/嗑药 |
| 体质 | 疑难杂症 | 你的痛苦感知 | 描述身体不适 |
| 运动 | 灵活 | 你的反射本能 | 提醒你危险信号 |
| 运动 | 权威 | 你的权力欲 | 命令/威胁/恐吓语气 |

### 设计原则
1. **机制即叙事**：技能检定不是"能不能做到"，而是"你内心的哪个声音更强"
2. **失败也是叙事**：检定失败不是Game Over，而是另一种有趣的展开
3. **无战斗系统**：所有冲突通过对话和技能检定解决
4. **思维阁**：玩家"装备"意识形态，影响技能加成和对话选项

### 对话结构
```
[普通对话文本]
    ← 技能检定分支 [百科全书·成功]: [检定通过的文本]
    ← 技能检定分支 [百科全书·失败]: [检定失败的文本]
    ← 被动技能触发 [内陆帝国]: [无需检定，自动触发的内心独白]
[玩家选择]
    → 选项A (需要修辞学6)
    → 选项B (需要权威4)
    → 选项C (无需检定)
```

### 叙事数据量
- 对话数据库：39MB SQLite
- 估计总字数：约100万字（英文）
- 主角没有配音，全靠文字呈现内心世界

### 对游戏策划的启示
- **叙事深度不依赖3A画面**——纯文字也能创造顶级体验
- **技能系统可以服务叙事**而非只服务战斗
- **让失败变得有趣**是叙事设计的最高境界
- **检定系统**可以借鉴到SLG的外交/谈判/审讯等场景
