## 📖 项目简介

三国志3复刻版是一款经典策略游戏《三国志3》的现代Web复刻版本。项目采用 React 18 + TypeScript 开发，使用 Canvas 绘制精美的六边形地图，实现了完整的内政、军事、外交、武将等核心游戏系统。

### 🎮 游戏特色

- **经典复刻**：忠实还原三国志3的核心玩法和策略深度
- **现代技术**：采用 React 18 + TypeScript + Vite 现代前端技术栈
- **精美画面**：六边形地图系统，古典中国风格UI设计
- **完整系统**：内政、战争、武将、外交、AI等20+核心系统
- **AI对战**：智能AI系统，支持多种难度和君主性格

## ✨ 功能特性

### 🗺️ 地图系统
- 六边形网格地图
- 10个州区域划分（司隶、雍州、益州等）
- 4种地形类型（平原、山地、水域、关隘）
- 地图缩放、拖拽、居中功能
- 城市连线显示

### 🏛️ 内政系统
- 城市发展（人口、商业、农业）
- 资源管理（金钱、粮草、士兵）
- 设施建设（市场、农场、兵营、城墙）
- 季节变化影响
- 天气系统

### ⚔️ 战争系统
- **战斗类型**：野战、攻城战
- **阵型系统**：鱼鳞、鹤翼、长蛇等8种阵型
- **战术系统**：火攻、伏兵、鼓舞等12种战术
- **计谋系统**：离间、伪报、混乱等10种计谋
- **单挑系统**：武将单挑对决
- **士气系统**：士气影响战斗力
- **疲劳度系统**：行军和战斗消耗体力

### 👤 武将系统
- 100+历史武将
- 五维属性（武力、智力、政治、魅力、统率）
- 武将技能和特技
- 武将关系系统（义兄弟、仇敌、配偶）
- 官职系统
- 武将性格（刚胆、胆小、仁义、冷静、鲁莽、普通）
- 忠诚度系统

### 🎯 搜索系统
- 武将搜索
- 宝物搜索
- 人才发现
- 搜索成功率计算

### 🚚 后勤系统
- 粮草运输
- 补给线管理
- 粮草消耗计算
- 缺粮影响

### 🤖 AI系统
- **局势评估**：实力、威胁、机会、需求评估
- **决策生成**：内政、军事、外交、武将决策
- **权重评分**：动态权重计算
- **君主性格**：6种性格影响AI决策
- **难度调节**：初级、中级、高级

### 🤝 外交系统
- 结盟、宣战、求和
- 离间计（降低敌方武将忠诚度）
- 贿赂、威胁
- 劝降武将/势力
- 外交关系值管理

### 🕵️ 间谍系统
- 派遣间谍
- 情报收集
- 破坏活动
- 煽动叛乱
- 反间谍

### 📜 历史事件系统
- 历史事件触发
- 随机事件
- 灾害事件（旱灾、瘟疫）
- 吉祥事件（丰收、人才现世）

### 💾 存档系统
- 10个存档槽位
- 自动存档
- 导出/导入存档
- 存档元数据管理

### 🎵 音效系统
- UI音效
- 战斗音效
- 事件音效
- 音量控制
- Web Audio API生成音效

### ⚡ 性能优化
- FPS监控
- 虚拟列表
- 节流防抖
- 渲染缓存
- 低性能模式自动切换

## 🚀 快速开始

### 环境要求

- Node.js >= 16.0.0
- npm >= 7.0.0 或 yarn >= 1.22.0

### 安装步骤

```bash
# 克隆仓库
git clone https://github.com/yourusername/sgz3.git

# 进入项目目录
cd sgz3

# 安装依赖
npm install
# 或
yarn install

# 启动开发服务器
npm run dev
# 或
yarn dev

# 构建生产版本
npm run build
# 或
yarn build

# 预览生产版本
npm run preview
# 或
yarn preview
```

### 访问游戏

开发服务器启动后，访问 `http://localhost:5173` 即可开始游戏。

## 🛠️ 技术栈

### 前端框架
- **React 18.3** - 现代化UI框架
- **TypeScript 5.5** - 类型安全
- **Vite 8.0** - 快速构建工具

### 状态管理
- **Zustand** - 轻量级状态管理

### 样式方案
- **Tailwind CSS 3.4** - 原子化CSS框架
- **PostCSS** - CSS后处理器

### 图形渲染
- **Canvas API** - 地图和战斗渲染
- **六边形网格** - HexGrid工具类

### 音频处理
- **Web Audio API** - 音效生成

### 数据存储
- **LocalStorage** - 存档存储
