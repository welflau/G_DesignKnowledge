## 正文
<h2 align="center">UI</h2>
<p>
当下的游戏，玩家主要的（高频的）交互方式有：跟3D场景中的内容进行交互、跟UI进行交互、跟外接设备进行交互。在很多类型游戏中，玩家主要的交互对象就是UI，它是支撑游戏体验的重要环节。
</p>

**关键词:**<br/>
*UI,用户界面,锚点,Anchor,对齐,九宫格,控件,MVC,图集,字体,Font,HUD,输入法,IME,层级,排版布局,文字排版,动画,Tween,帧动画,按钮,进度条,列表,Checkbox,标签,文本框,窗体,对话框,模态化,页面,Sprite,拉伸,Tiled,事件,交互事件,矢量字库,位图字库,TTF,分辨率,安全区,触摸屏,手势,陀螺仪,马达,手柄,键盘,鼠标,多语言支持,AI Coding*

## 动画类型

**是什么？在哪用？**
| 维度 | 内容 |
|------|------|
| **作用** | 不同场景适用不同动画 |
| **应用场景** | 平滑过渡；复杂效果 |

| 类型 | 说明 | 优点 | 缺点 |
|------|------|------|------|
| Tween | 补间动画 | 性能好、易控制 | - |
| 帧动画 | 逐帧播放 | 表现力强 | 资源占用大 |

**会遇到哪些问题？用什么解决？**
| 问题 | 解决方向 |
|------|----------|
| **动画性能** | 硬件加速；优化计算；限制数量；对象池 |
| **不流畅** | 优化计算；固定步长；减少复杂度；缓动函数 |

| 要点和思考方向 |
|----------------|
| Tween 适合简单过渡；帧动画适合复杂效果；注意性能 |

| 类型 | AI Coding 指南 |
|------|----------------|
| **交互提示** | 说明动画类型、性能；如「弹窗动画」「按钮反馈」「卡顿」 |
| **方法** | Tween 用 DOTween/LeanTween；帧动画用 Sprite 序列；缓动函数 |
| **应用** | 入场出场；交互反馈 |
| **提示词范例** | 「弹窗打开用缩放+淡入动画，请用 Unity DOTween 或 Animation 实现，并说明 ease-out 与 ease-in-out 的选择」 |
