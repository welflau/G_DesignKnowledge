# 基建系统概述

## 系统说明

基建系统是明日方舟中的核心养成系统之一，玩家可以在罗德岛基地中建造各种设施来生产资源、培养干员。

## 基建设施类型

1. **控制中枢**: 提升基地整体效率
2. **发电站**: 提供电力支持
3. **制造站**: 生产各类材料
4. **贸易站**: 将赤金转化为龙门币
5. **宿舍**: 恢复干员心情
6. **会客室**: 线索交流
7. **加工站**: 加工材料
8. **办公室**: 公开招募
9. **训练室**: 训练干员技能
