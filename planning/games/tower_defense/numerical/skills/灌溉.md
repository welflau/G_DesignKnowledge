# 灌溉

## sktok_lhhe

- 技能类型: 自动触发
- SP回复类型: 自动回复
- 持续时间: 2.0秒

**效果描述:** 仅可部署在地图边缘格或其他【河流】周围四格。周围其他【地形】的效果提高。周围四格有地面时，会灌溉土壤，形成【沃土】并在技能开启时生成潜水员（击败获得狙击进阶信物和特种进阶信物）或码头水手（击败获得近卫进阶信物和先锋进阶信物）
