# 情感与惊喜设计

> 来源：《任天堂的体验设计》第2章

## 心理饱和
- **来源**: 《任天堂的体验设计》第2章
- **核心**: 相同刺激重复多次后大脑反应会逐渐减弱，这是直觉设计的致命弱点。
- **详述**: 心理学中的"心理饱和"（或"心理适应"）指的是：当相同刺激反复出现时，大脑的反应会逐渐变弱。在游戏中，直觉设计循环（假设→尝试→高兴）虽然能让人感到有趣，但"假设"和"尝试"阶段会给玩家带来不安——不知道假设是否正确的不安、尝试时不确定是否对的不安。当直觉设计反复出现时，玩家的内心在"不安"与"高兴"之间不断摇摆，疲惫不可避免。

  更重要的是，无论多么精心的直觉设计，同样的体验持续多次就会令人厌烦，这是无法避免的。《勇者斗恶龙》注定是一款需要学习大量内容的游戏，玩家会陷入"想玩，但很困"的奇怪状态——仍有继续玩的欲望，但疲劳和厌倦已在身心累积。
- **适用场景**: 设计长时间游戏体验时，必须预判玩家的疲劳和厌倦累积点，并在该节点前安排惊喜设计。
- **示例**: 《勇者斗恶龙1》中玩家必须学完全部8个指令才能完成初期探索，这是密集的直觉设计阶段。设计师在玩家学完所有指令后安排了"惊喜刺激"登场，精准对应疲劳和厌倦的顶峰时刻。

## 惊喜设计循环（误解→尝试→惊讶）
- **来源**: 《任天堂的体验设计》第2章
- **核心**: 让玩家自发建立错误假设、尝试后发现错误而大吃一惊，借此消除疲劳和厌倦。
- **详述**: 惊喜设计的结构是直觉设计的镜像反转：1.误解——玩家自发做出"要做××吗？"的错误假设，并坚信假设正确；2.尝试——玩家自发尝试"试一试××"并行动，同时坚信自己是对的；3.惊讶——玩家自发发现"××是错误的"，大吃一惊。

  实施惊喜设计需要三个步骤：首先，掌握玩家疲劳和厌倦的时机（如《勇者斗恶龙》中"学完8个指令时"）；其次，事先构建让玩家产生误解的游戏主题（花时间让玩家慢慢学习"这个游戏很严肃"的错误认知）；最后，设计能"暴露"误解的剧情，让玩家对前提的坚信（"这个游戏是××"）和对日常的坚信（"禁忌不会出现"）同时落空。

  惊喜设计是为消除连续直觉设计带来的疲劳和厌倦而存在的"必要之恶"，它让游戏从"能玩"进化到"能玩下去"。
- **适用场景**: 直觉设计循环持续后玩家出现疲劳征兆时，需要穿插惊喜设计来重置玩家状态。
- **示例**: 《勇者斗恶龙3》最终阶段，玩家进入的"异世界"竟然是《勇者斗恶龙1》的世界——完全颠覆了玩家的预期。龙王在最后关头提出"我把世界的一半给你"的诱惑，背叛了"勇者必战胜魔王"的前提。

## 对前提的坚信与背叛
- **来源**: 《任天堂的体验设计》第2章
- **核心**: 利用玩家"这个游戏是××"的坚信，有计划地构建误解再将其颠覆。
- **详述**: 惊喜设计利用两种"坚信"来创造出乎意料的体验。第一种是对前提的坚信——"这个游戏是××"。设计师需要花很长时间不断铺垫，让玩家在反复学习中形成对游戏性质的牢固认知，然后在关键时刻颠覆它。

  让玩家产生误解是相当困难的工作。为让对方对前提产生误解，就必须花很长时间不断"说谎"。让玩家对前提的坚信反复落空，等于动摇游戏的根基。即使在铺垫过程中，玩家的疲劳和厌倦也在累积。

  正因为如此，《勇者斗恶龙》虽然被称为"教科书式"游戏，实际上恰恰是"非教科书式"的——它完全背叛玩家的预期。那些被认为经典的剧情转折，本质上都是精心设计的谎言和真相揭晓。
- **适用场景**: 长篇幅游戏中需要重大剧情转折或体验高潮时。
- **示例**: 《勇者斗恶龙1》中，设计师从游戏开始就费尽心思营造"严肃的冒险故事"氛围，让玩家深信这是一款庄重的RPG。当"惊喜刺激"突然出现时，严肃前提被打破，玩家的惊讶感因此被放大。

## 对日常的坚信与禁忌
- **来源**: 《任天堂的体验设计》第2章
- **核心**: 打破"禁忌不会出现在日常生活中"的信念，用禁忌主题创造轻松的惊讶。
- **详述**: 第二种坚信是对日常的坚信——"应该不会出现禁忌"。在平凡的日常生活中不应该出现的东西被称为禁忌。玩家认为禁忌一定不会在游戏中出现，这种信念是设计体验的宝库：只要让禁忌登场，就能让玩家出乎意料。

  与颠覆前提相比，利用禁忌主题是更轻松实现惊喜的方法。不需要花长时间构建误解，只需打破对日常的坚信即可。虽然惊讶程度较小，但有一定的缓解疲劳和厌倦的效果。设计师可以放心地"欺骗"玩家，因为遇到禁忌内容的玩家不会生气，反而会开心。
- **适用场景**: 需要快速制造轻松惊讶、缓解玩家疲劳时，或者无法安排长时间铺垫的场景。
- **示例**: 《勇者斗恶龙》系列每部都会安排"惊喜刺激"场景，它们总是在玩家高度紧张冒险之后出现——勇者本应在拯救世界，却被NPC邀请"来点惊喜刺激"，日常禁忌的突然出现让人忍俊不禁。

## 禁忌主题（1-4）：食、性、得失、认可
- **来源**: 《任天堂的体验设计》第2章
- **核心**: 利用人类出于本能追求的积极主题来扰乱平静心绪，消除疲劳和厌倦。
- **详述**: 10种禁忌主题中的前4种是一组每个人都出于本能追求的积极主题：食、性、得失、认可。这些主题直接触动人类最根本的欲望——饥饿、吸引、赢取、被认同。正因为它们如此本能，所以几乎所有人都无法对其保持平静。

  在电子游戏发展初期，高难度麻将游戏在每关之间插入图片，正是利用了"性"这类禁忌主题来驱动玩家持续游戏。引入"食""得失""认可"等主题扰乱人们的平静心绪，可以有效缓和长时间玩游戏产生的疲劳和厌倦。

  体验设计的指标是："这个体验，描绘的是人类本能的欲望吗？"反过来说，要想找到完全没有这些主题的内容是极其困难的。只要内容以"体验到最后"为目标，禁忌主题都有可能出现。
- **适用场景**: 设计持续体验时，需在适当节点插入触动本能欲望的内容来激活玩家。
- **示例**: 麻将游戏每关之间的奖励图片利用"性"主题；《勇者斗恶龙》中的赌场利用"得失"主题——玩家赢取金币换取强力装备，暂时脱离严肃冒险，在赌博中获得本能满足后再自然回归主线。

## 禁忌主题（5-8）：肮脏、暴力、混乱、死亡
- **来源**: 《任天堂的体验设计》第2章
- **核心**: 利用人们本能想要回避的消极主题制造强烈冲击，打破玩家的心理惯性。
- **详述**: 第5到第8种禁忌主题是所有人都讨厌的消极主题：肮脏、暴力、混乱、死亡。在所有剧情中，反派角色都是不可或缺的——他们旁若无人地使用暴力、引发混乱、带来疼痛和死亡。

  设计师必须舍弃自己人格可能被怀疑的不安心理，有意识地使用这些消极主题。所有游戏设计都是为了让体验的玩家感到惊讶，让游戏体验持续下去。体验设计的指标是："这个体验，有人们想要回避的东西吗？"

  消极主题之所以有效，是因为它们打破了玩家在日常生活中维持的心理平衡。当玩家在安全的游戏环境中遭遇这些主题时，产生的不是真正的恐惧，而是一种"安全的刺激"——这种刺激恰恰能有效消除累积的疲劳和厌倦。
- **适用场景**: 游戏体验变得平淡、玩家注意力开始分散时，用消极主题制造冲击。
- **示例**: 《勇者斗恶龙》系列中魔王正在进行惨无人道的杀戮——暴力与死亡主题让玩家产生紧迫感和恐惧感，这些消极情绪反而让冒险体验更加深刻，也为后续的惊喜设计提供了更强烈的对比。

## 禁忌主题（9）：侥幸与偶然
- **来源**: 《任天堂的体验设计》第2章
- **核心**: 利用人们对概率和运气的迷恋来消除疲劳，创造"祈祷"的体验。
- **详述**: "侥幸"是游戏业界对"悬赏、抽签、有没有掉钱"这类不知不觉追求幸运心情的称呼。侥幸与偶然之所以强大，是因为它触及了人类对不确定性的本能迷恋——我们无法抗拒"说不定会走运"的念头。

  《勇者斗恶龙4》引入了赌场系统，这是设计师有意识的决断：勇者本应去拯救世界，却泡在赌场里玩老虎机和扑克。赌场有意识地剥夺了认真冒险的玩家的严肃感，让他们暂时停止冒险。等玩家在赌场玩够了、手上有了强力装备后，设计师再让他们自然地踏上冒险之旅。游戏中的赌场不会像真正的赌场那样让玩家输——它会以不引起怀疑的程度让玩家获胜。

  战斗中的"会心一击"也是对概率的运用：极少出现的强烈攻击让玩家特别开心，即使被逼到败北边缘，只要使出会心一击就有可能反败为胜。
- **适用场景**: 需要让玩家暂时脱离主线体验、同时为后续冒险积蓄动力的场景。
- **示例**: 《勇者斗恶龙4》的赌场让玩家在20~30小时的长篇冒险中暂时停下来赌博，赢取装备后再带着好心情和强力道具回归主线。战斗中的"会心一击"在均衡分配各种要素的同时消除疲劳。

## 禁忌主题（10）：个人隐私
- **来源**: 《任天堂的体验设计》第2章
- **核心**: 暴露玩家的内心世界和个性偏好，创造最强烈也最私人的惊讶体验。
- **详述**: 最后一种禁忌主题是"个人隐私"。在日常生活中，大家都隐藏着个人隐私，如果被人知道了就无法平静。正因为人们认为"日常生活中必定隐藏着个人隐私"，所以当个人隐私被暴露时会带来最强烈的惊喜。

  《勇者斗恶龙》中输入勇者名字的简单画面留下了深刻印象——有人用自己的名字觉得怪怪的改了，有人把喜欢的人名字加进去不能让朋友看到，有人起了不雅的名字。这个看似素净的画面暴露了取名品位这一隐私。

  《勇者斗恶龙5》中玩家必须从两位女性中选择一位结婚，这个选择充分暴露了玩家对另一半的喜好。玩家之间热烈讨论"你和谁结婚了"，至今粉丝仍在争论——这正是因为个人隐私主题带来了无可替代的体验强度。

  体验设计的指标是："这个体验能体现出个性吗？"越是暴露个人隐私的内容越让人惊讶。
- **适用场景**: 需要创造强烈的个人化体验、让玩家对游戏产生深层情感连接时。
- **示例**: 《勇者斗恶龙5》的结婚选择让玩家在比安卡（青梅竹马、好胜）和芙萝拉（富家千金、文静）之间抉择，充分暴露了个人偏好；《勇者斗恶龙》的勇者命名画面让玩家的取名品位无处可藏。

## 直觉与惊喜的交替节奏
- **来源**: 《任天堂的体验设计》第2章
- **核心**: 直觉设计让人理解学习，惊喜设计消除疲劳厌倦，两者交替排列构成体验的基本结构。
- **详述**: 内容的基础是直觉设计和惊喜设计的组合。对于短时间体验的内容（广告、漫画、新闻），惊喜设计会出现在开始随后密集出现，一边吸引注意力一边传达信息。对于需要长时间体验的内容（电影、游戏），设计师会以直觉设计开始，在直觉设计持续过程中抓住机会穿插惊喜设计。

  猎奇影像只指向让人出于本能兴奋的内容，里面充满禁忌主题。而《勇者斗恶龙》这类经典RPG，设计师以直觉设计开始（教玩家学习指令系统），在玩家疲劳时穿插惊喜设计（"惊喜刺激"登场），再回归直觉设计（继续冒险），循环往复。

  正如任天堂已故董事长岩田聪所说："游戏不是生活必需品，所以需要产生惊喜。"玩家凭直觉玩游戏，游戏也出乎玩家意料让其惊喜——这两种截然相反的体验交织在一起，让游戏业得以存续。游戏从"能玩"发展到"能玩下去"，靠的就是在连续直觉设计中穿插惊喜设计。
- **适用场景**: 规划整体游戏节奏、设计关卡顺序和剧情结构时。
- **示例**: 《勇者斗恶龙》系列每部作品都在"学习新系统→冒险战斗→疲劳累积→禁忌惊喜→重获动力→继续冒险"的循环中推进。设计师堀井雄二精准把握每个循环的节奏，让RPG这一"有趣但难以令人理解"的游戏类型被日本大众广泛接受。

## 兴趣曲线——体验质量的可视化度量
- **来源**: 《游戏设计艺术》第16章/透镜69
- **核心**: 任何娱乐体验的质量都可用"兴趣曲线"来衡量——优秀的体验遵循"钩子→逐步攀升→高潮→余韵"的形状，在三个层级上碎片化存在（整局游戏、每一关、每次挑战）。
- **详述**: 兴趣曲线的典型形态：初始兴趣(A)→钩子(B，快速抓住注意力)→逐步提升的兴趣伴随间歇性回落(C,D,E,F)→高潮(G)→满意的结束(H)。失败的曲线则缺少钩子、兴趣持续走低直到跌破"兴趣临界点"（玩家关掉游戏），即使后来有有趣的事也没用。兴趣曲线可以在三个层级碎片化存在：整款游戏的开场→关卡递进→最终Boss；每一关的新鲜感→挑战递进→关卡Boss；每次挑战的有趣介绍→阶梯式提升的困难。为不同人群设计不同的兴趣曲线是有益的练习。
- **适用场景**: 评估关卡/剧情/整体体验的节奏是否合理时；诊断"玩家为什么在某个位置弃游"时。
- **示例**: 《半条命2：第一章》中玩家死亡次数按关卡排列形成的曲线完美匹配经典兴趣曲线——开场低、逐步攀升、高潮密集、结尾回落。

## 体验的三大兴趣要素
- **来源**: 《游戏设计艺术》第16章/透镜70、71、72
- **核心**: 体验的趣味由三个因素构成：内在兴趣（事情本身就引人注目）、诗意/美感（呈现方式的艺术质量）、投影/代入（客人将自己投射进体验中）——三者互补，缺什么都可以用另外两个弥补。
- **详述**: 内在兴趣来自冒险、异常、戏剧性变化——人与鳄鱼摔跤比人吃三明治有趣。美感来自音乐、视觉、叙事的艺术品质——小提琴协奏曲缺乏内在兴趣但音乐之美足以支撑体验。投影来自共鸣和想象——俄罗斯方块的积木块既无内在兴趣也无美感，但客人做出所有决定的强烈投影弥补了一切。互动娱乐在投影上有天然优势——玩家是主角，事件直接发生在他们身上。这三种因素的组合可以在不同时刻变化——画出每种因素随时间的贡献曲线，就能精确诊断哪些时刻"不够有趣"以及如何弥补。
- **适用场景**: 分析游戏体验中"无趣"时刻的成因并寻找弥补方案时。
- **示例**: 杂耍电锯（高内在兴趣+低美感+中等投影）、小提琴协奏曲（低内在兴趣+高美感+低投影）、俄罗斯方块（低内在兴趣+低美感+极高投影）——三者都能创造精彩体验，只是配方不同。

## 值得纪念的时刻是体验的骨架
- **来源**: 《游戏设计艺术》第16章/透镜68
- **核心**: 任何娱乐体验都是一系列"时刻"，其中最有力量的时刻构成兴趣曲线上的星星——首先识别最重要的时刻，然后将每个时刻做到尽可能强大。
- **详述**: 在迪士尼幻想工程，向CEO展示新设施时必须回答"你的体验中最佳十个时刻是什么"——如果没有清晰答案，提案就结束了。识别重要时刻是第一步，强化它们是第二步。每个重要时刻应该拥有强烈的内在兴趣、优美的呈现和深刻的玩家代入。当你在设计关卡或剧情时，先列出"玩家会记住的5-10个时刻"，然后逐一审视：这个时刻够强吗？如何让它更强？如果砍掉某个时刻不会削弱整体体验，那它可能不够重要。
- **适用场景**: 关卡设计初期确定"高光时刻"、剧情关键节点规划、Boss战设计时。
- **示例**: 《半条命2》的"最佳十个时刻"包括：开场列车到达17号城、第一次获得重力枪、操纵蚁狮大军进攻监狱等——每个都是玩家会记住数年的瞬间。
