# 关卡设计

> 来源：《游戏设计的236个技巧》第3章

## 游戏循环设计（学习→尝试→应用→精通）
- **来源**: 《游戏设计的236个技巧》第3章/3.1节
- **核心**: 一个关卡应由"学习""尝试""应用""精通"四个阶段构成，多层次化后可让玩家百玩不厌。
- **详述**: 宫本茂曾表示，《马里奥》的任何一个挑战要素都一定具备学习的场所、实际尝试的场所、应用的场所和练至精通的场所。以场景1-1为例：遇到第一个敌人的地方是"学习"（学到踩踏消灭敌人）；前半段是"尝试"（使用学到的动作）；后半段更大的缺口和强敌是"应用"；终点旗杆则是"精通"（从高台跃起抱住旗杆顶端可获得最高奖励）。将这四个步骤多层次化——一个场景本身是"学习"，场景结构中也存在"学习→尝试→应用→精通"的循环——就能创造出让人想玩到精通的游戏。
- **适用场景**: 任何动作类、探索类游戏的单关卡设计，尤其是希望让玩家掌握新机制时。
- **示例**: 《超级马里奥3D大陆》场景1-1中，玩家从学会过河、踩敌人、顶砖块，到最终抱住旗杆顶端获得烟火奖励，完整经历四个阶段。

## 节拍·节奏·步调
- **来源**: 《游戏设计的236个技巧》第3章/3.1节
- **核心**: 创造关卡的过程就是创造供玩家享受的节拍节奏（Groove），设计游戏不只是设计地图形状和难度，更是对游戏节拍与节奏的设计。
- **详述**: 台阶状方块的落差和间距决定了跳跃的节拍节奏；等间隔的铁球让玩家找到律动感；给看似随机摆动的铁球留一个"可以一口气冲过去"的时机，发现绝妙时机能让乐趣成倍增长。缺口的难度排列也构成节拍——由易到难让玩家积蓄成功体验，有节奏地排列则可让玩家一气呵成通关。步调则控制游戏推进速度：马里奥基本由玩家掌控步调，但强制滚动场景中步调由游戏方控制。一款有趣的游戏离不开"节拍""节奏""步调"三个要素。在动作类游戏中，如果说玩家是乐器，那么关卡设计就是乐谱。
- **适用场景**: 所有动作类游戏的关卡设计，尤其是平台跳跃游戏和战斗场景。
- **示例**: 《超级马里奥3D大陆》中均匀间距的台阶让玩家从跳跃节拍中获得享受；《战神Ⅲ》中波浪式攻击用节拍节奏不断挑高玩家情绪。

## 不甘心的关卡设计
- **来源**: 《游戏设计的236个技巧》第3章/3.1节
- **核心**: 让玩家同时判断两个以上要素从而因慌乱失误，激发"不甘心"情绪，促使其重新挑战。
- **详述**: 在动作游戏中，如果完成一个动作需要同时判断两个或两个以上的要素，玩家往往会由于慌乱而失误。比如玩家在拿道具时被敌人追击，搞错起跳时机撞到敌人；或在控制跷跷板轨道车的同时还要躲避敌人。这种失误能激发"下次一定成功"的期待感与紧张感。随着游戏推进，需要同时做的事越来越多：从"两件事"到"不从旋转地面跌落+躲避敌人+跳到下一个旋转地面"的三件事。但关键在于必须让玩家能明确判断失败原因，否则不甘心会被窝火代替。
- **适用场景**: 动作类游戏中需要提升紧张感和重复挑战动力的场景。
- **示例**: 《超级马里奥3D大陆》场景1-4中，玩家在控制轨道车的同时处理敌人和道具；场景4-3中同时面临三个任务。

## 刻意安排的偶然发现
- **来源**: 《游戏设计的236个技巧》第3章/3.1节
- **核心**: 设计者在关卡中精心安置"偶然发现"，让玩家体验发现的惊喜和"只有自己发现了秘密"的优越感。
- **详述**: 偶然发现分为两种："刻意安排的"和"真正偶然的"。前者如用狸猫马里奥尾巴攻击敌人时破坏石块发现金币，看似偶然实为设计者精心布置；后者如爬树跳上管道发现隐藏区域，需要探索欲较强才能发现。无论哪种，偶然发现都是玩家"忍不住和别人分享的秘密"，构成了游戏外的交流。刻意安排的偶然发现是激发玩家真正探索欲的诱饵。
- **适用场景**: 探索型游戏、平台动作游戏中激发玩家探索欲望。
- **示例**: 《超级马里奥3D大陆》中，破坏石块发现金币（刻意安排）；爬上树发现管道内部隐藏区域（真正偶然）。

## 与玩家构建信任关系的关卡设计
- **来源**: 《游戏设计的236个技巧》第3章/3.1节
- **核心**: 关卡设计作为与玩家对话的手段，信任关系的核心在于：能取得报酬的地方绝不会遇到莫名其妙的死法。
- **详述**: 金币在超级马里奥中不仅是报酬，更是信任关系的象征。金币可用作路线提示（提供导航），空中的金币可暗示安全落点。在马里奥中绝对不会出现"拿这个金币肯定会死一次"的设计，因为这种设计只要出现一次，玩家与游戏的信任关系就会分崩离析。为实现对话，开发者在设计关卡时必须在脑海中想象玩家进行游戏时的情景，预想多种玩法风格，推测通关所需时间，然后通过关卡设计与未来的玩家对话。
- **适用场景**: 所有需要反复挑战的游戏，特别是有惩罚机制的动作游戏。
- **示例**: 《超级马里奥3D大陆》中，有金币的地方一定安全，空中金币暗示可以平安落回地面。

## 乘法的关卡设计
- **来源**: 《游戏设计的236个技巧》第3章/3.2节
- **核心**: 将关卡设计与故事、互动要素相乘而非简单相加，创造"关卡设计×故事"的动态体验。
- **详述**: 以往动作类游戏是"关卡设计+故事+关卡设计"的加法设计。《战神Ⅲ》将故事融入玩家动作中，创造"关卡设计×故事"的乘法设计。其实现手法是"根据情境进行战斗的关卡设计"：盖亚的身体在攀登过程中不断运动，在不同情境下可视为不同的"通道""房间·广场"，连接起来形成不同关卡设计。在互动层面也用"探索×动作""动作×谜题"等形式实现相乘效果。如拉动杠杆后平台移动但会自动归位，将谜题与动作结合——解开谜题的过程中对动作的摸索也十分有趣，能让玩家百玩不厌。
- **适用场景**: 叙事驱动的动作游戏、需要将多种玩法融合的关卡设计。
- **示例**: 《战神Ⅲ》中在巨人盖亚身体上的战斗，情境动态变化创造了"互动的故事"。

## 路径探索的构造
- **来源**: 《游戏设计的236个技巧》第3章/3.2节
- **核心**: 路径构造分为单线型、分枝型、网状，分岔口的设计决定探索乐趣的质与量。
- **详述**: 单线型不会迷路，适合在其上设置战斗或谜题。分枝型在主干道上添加分支（均为死路），玩家会短时间迷路但能快速回归。网状结构自由度高但容易迷路，属于非直线行进型关卡。分岔口根据功能分为：死胡同（激发探索欲，设报酬可增强后续探索意愿）、简单障碍（需寻找道具开门）、区域阻隔（连接尚未开放区域的阀门）、捷径（供快速返回）、完全分岔口（通向不同区域，需平衡难度）。路径探索还包含"动作选择"——用爬墙、攀天花板等动作探索路径能激发本能的危机意识和紧张感。
- **适用场景**: 探索型动作游戏、地下城设计、冒险游戏。
- **示例**: 《战神Ⅲ》中绳索动作和抓钩跳跃在路径探索中同时考验动作技巧，失误可导入岔路。

## 封闭式战场与开放式战场
- **来源**: 《游戏设计的236个技巧》第3章/3.2节
- **核心**: 割草类游戏需要用"魔法屏障""栅栏"封闭战场营造紧迫感，战场形状决定战术。
- **详述**: 封闭式战场分为：圆形/矩形型（如角斗场，中央利于攻击但也容易被围攻）、通道型（可形成夹击，被逼至一端则闪避受限）、有障碍物型（产生攻击死角，障碍物也可作盾牌但可能成为镜头死角）。天花板高度也限制玩法——割草类游戏需要为浮空攻击留出高度，因此天花板往往极高。通过改变战场形状，即使设置相同敌人也能诱导玩家使用不同战术。此外，将"机关×战斗""谜题×战斗"等组合，可进一步丰富战场设计。
- **适用场景**: 割草类动作游戏的战斗场景设计。
- **示例**: 《战神Ⅲ》中赫拉克勒斯BOSS战前有数波杂兵的波浪式攻击，黑暗场景需用照明与敌人战斗，完全改变战术。

## 波浪式攻击创造节拍节奏
- **来源**: 《游戏设计的236个技巧》第3章/3.2节
- **核心**: 敌人的波浪式攻击可像音乐一样控制战斗的节拍与节奏，是关卡设计中营造张弛感的核心手段。
- **详述**: 波浪式攻击的敌人出现方式分三种：分阶段出现（消灭当前敌人后出现下一波，节奏有张弛）、根据被消灭数量出现（战场始终有敌人，紧张感不中断）、经过一定时间出现（节奏完全由关卡设计方掌控）。节拍由敌群出现的速度表现，节奏由敌群的人数和种类决定，就像音乐中的音程和音色。将战场从单纯战斗场所上升为享受音乐般节拍节奏的场所后，会发现关卡设计中隐藏的另一番乐趣。
- **适用场景**: 动作游戏中需要控制战斗节奏的BOSS战或大型战斗场景。
- **示例**: 《战神Ⅲ》赫拉克勒斯BOSS战，先由杂兵发动数波攻击（节奏逐渐加快），然后赫拉克勒斯登场。

## 3D关卡设计的方向性
- **来源**: 《游戏设计的236个技巧》第3章/3.3节
- **核心**: 通过地标、宽度、高度、亮度、图腾柱等手段明确地图方向性，让玩家在脑中描绘出地图。
- **详述**: 3D游戏中玩家可自由操纵镜头，地图方向性与画面方向性不再一致，容易迷失。最简单的方法是设置"地标"——一个地图中仅存在一个的特别物体，必须高大显眼保证从任何地方都能看到。相对方向性可用道路"宽窄"（越来越窄暗示接近终点）、"高低差"（坡道或台阶，不受近大远小影响）、"明暗"（敞亮vs葱郁的小路）来表现。"图腾柱"（雕像、石像、门等）最能创造方向性且可表现场所意义。路牌属于文字信息，不如图腾柱一目了然。关键在于将视觉信息与体感记忆的信息组合，用身体直接记忆的关卡设计比用脑袋处理的更有效。
- **适用场景**: 以探索为主的3D游戏、地下城设计。
- **示例**: 《塞尔达传说：天空之剑》菲罗奈森林中，神殿作地标，道路有宽窄和高低差，鸟型纪念碑作图腾柱。

## 关卡设计的诱导
- **来源**: 《游戏设计的236个技巧》第3章/3.3节
- **核心**: 通过道路形状、门的开关、墙壁的洞、岔路差异等微妙设计自然诱导玩家，让玩家觉得是自己选择发现了道路。
- **详述**: 宽路暗示主线，窄路暗示隐藏秘密；笔直的路给人最快抵达目的地之感，蜿蜒的路带来迷惑和神秘感。人类下意识想选择的路线称为"希望线"，可在希望线上设置陷阱或敌人给玩家惊喜。门的大小给玩家不同预感——大门让玩家觉得"门后有东西"；开着的门给人被欢迎之感并引导迷路的玩家；关着的门带来被拒绝感但激发探索欲。将门的功能加入互动可创造"能玩的门"（可破坏的蜘蛛网、可炸开的岩石、可推下的圆木），兼作单行道或捷径。岔路中，只要分岔路形状或宽度各不相同，玩家就会自发联想并做出判断。
- **适用场景**: 探索型游戏中的路径设计、引导玩家但不破坏自由感。
- **示例**: 《塞尔达传说：天空之剑》中三扇开着的门引导迷路玩家前进；推下圆木创造捷径，将封闭式探索转化为开放式探索。

## 封闭式探索与开放式探索的融合
- **来源**: 《游戏设计的236个技巧》第3章/3.3节
- **核心**: 在同一张地图中巧妙融合封闭式探索（按既定顺序体验）与开放式探索（自由探索），可创造丰富细腻的关卡设计。
- **详述**: 封闭式探索中玩家按既定顺序和路径进行，机关、动作、解谜按部就班。开放式探索不限制自由，以"寻人""寻物"等不需固定顺序的玩法为主，可中途放弃随时重来。大部分探索型游戏将两种探索分得很明显（如世界地图=开放式，地下城=封闭式）。《塞尔达传说：天空之剑》则将两者融入同一张地图：前半部分是诠释既定路径的封闭式探索，后半部分突然改变节拍让玩家自由探索。随着玩家不断设置圆木开通捷径，前半的封闭式探索地图最终也变为开放式探索地图。
- **适用场景**: 大型探索场景、需要兼顾叙事引导和自由度的游戏。
- **示例**: 《塞尔达传说：天空之剑》菲罗奈森林，前半按顺序触发事件，后半自由寻找库伊。

## 自发的游戏的乐趣
- **来源**: 《游戏设计的236个技巧》第3章/3.3节
- **核心**: 通过大量反应和报酬回应玩家"试试看！找找看！"的期待，催生玩家自发的发现乐趣。
- **详述**: 玩家在玩游戏时往往有与剧情无关的想法，如"这个动作如果能这么用就有意思了"。为实现这些想法，游戏中需在草、树木等日常元素上添加大量互动元素，并提供卢比或道具作为报酬。由于互动元素与剧情无直接关联，玩家会在其他场所也进行类似尝试。诱导玩家使用特殊动作的关卡设计用过多会让玩家感到被牵着鼻子走，而"自发的游戏的乐趣"能让玩家乐此不疲地主动使用玩家角色动作。其核心构造是"自发的游戏的乐趣×发现"。
- **适用场景**: 高自由度探索游戏、需要让玩家自发尝试动作和道具的关卡。
- **示例**: 《塞尔达传说：天空之剑》中砍发光树木获取卢比、撞树获取果实、寻找女神方块开启隐藏岛屿。

## 开放世界的区域特色与高度层次
- **来源**: 《游戏设计的236个技巧》第3章/3.4节
- **核心**: 开放世界必须死守"场景间可以相似但不能相同"的规则，按区域设置特色并在不同高度运用不同关卡设计。
- **详述**: 如果整个世界到处都是相似构造，任务很难创造个性。为保证规则，需将场景分割成多个区域，每个区域设置特殊地标并添加特色。如公园路区有高层建筑（可从上空攻击+高处避难），保利区有拱廊天花板（不能从上空攻击也不能避难），娱乐区被海水淹没（操作失误会掉入海中）。蝙蝠侠可在纵向自由移动，因此"房顶""中层""地面""地下"不同高度有不同关卡设计。关卡设计的"密度"也必须与玩家的移动动作和速度相配套。开放世界的关卡设计并不是自由度够高地图够大就行的，所有场景都应是独一无二的。
- **适用场景**: 开放世界游戏的地图设计、大型探索场景。
- **示例**: 《蝙蝠侠：阿甘之城》中500+任务分布在不同特色区域，高度越高越安全（上空安全、地面危险）的设计忠实还原蝙蝠侠的世界观。

## 锁&钥匙系统
- **来源**: 《游戏设计的236个技巧》第3章/3.4节
- **核心**: 通过获得新工具/动作来解锁之前无法进入的区域，让玩家的发现能力成为开启更广阔世界的钥匙。
- **详述**: 玩家推进主线或提升等级后解锁工具（蝙蝠镖、遥控电击器、爆炸凝胶、冻结器等），借助这些工具可进入隐藏地区。主线任务中也为工具嵌入了"学习→尝试→应用→精通"的游戏循环。后期考验玩家对工具的组合使用——如用冻结器在水面制造落脚点再用蝙蝠镖牵引绳索移动。机关诱导玩家对工具做"乘法"，让玩家脑中浮现"把道具与动作组合是不是能通过之前过不去的地方"的想法。道具和动作的相乘用法能大幅扩展玩家的自由度和想象力，拓展"可能空间"。
- **适用场景**: 探索型游戏中拓展可探索范围的机制设计。
- **示例**: 《蝙蝠侠：阿甘之城》中将多种工具组合使用，开启隐藏区域；探索范围随工具增多而不断扩大。

## 潜行动作的关卡设计
- **来源**: 《游戏设计的236个技巧》第3章/3.4节
- **核心**: 潜行关卡需重点设计"敌人的巡逻路线""掩体""玩家的避难场所"三要素，并加入时间维度的变化。
- **详述**: 即便在封闭式探索中仍保持"越高越安全"的概念。制高点虽为优势位置但并非永远安全——敌人可射击破坏制高点。战斗取胜的关键不取决于敌人动作而掌握在关卡设计手中，玩家每次遭遇新战斗都必须观察战场并制定战术。潜行动作还需考虑"时间"要素——战场变化过程中必须出现破绽，给玩家留出敌人侦查死角的瞬间。这种随时间变化的关卡设计十分考验设计者实力。封闭式探索的关卡设计中留出足够高的自由度让玩家思考并尝试其他战术，可带来与开放世界截然不同的"开放式玩法"。
- **适用场景**: 潜行类游戏、需要战术思考的战斗场景。
- **示例**: 《蝙蝠侠：阿甘之城》中建筑内部潜行场景，需观察敌人巡逻路线利用通风口、可破坏墙壁、电子器械等创造突破机会。

## 空间关卡设计中给高度赋予意义
- **来源**: 《游戏设计的236个技巧》第3章/3.5节
- **核心**: 在不受重力束缚的游戏中，空间关卡设计必须给不同高度赋予不同意义（优劣势），否则游戏空间将成为一片茫茫宇宙。
- **详述**: 不受重力束缚的游戏使用空间关卡设计（而非平面关卡设计），允许玩家上下自由移动，需要用物理单位或游戏系统限制移动区域。但关键在于给高度赋予意义：受重力影响的地面部队使高处成为安全避难所；地面可作为连击击飞追加伤害的场所；低处障碍物更多适合用抓投攻击；还有只能在地面使用的武器。空间关卡设计中给高度赋予的意义决定着游戏手感和难度。如果不给高度赋予意义，左右游戏胜负的只是距离和攻击动作的优劣，感觉更接近射击类游戏。
- **适用场景**: 空战游戏、零重力/微重力游戏、飞行类动作游戏。
- **示例**: 《终极地带：引导亡灵之神》中地面部队、地面连击追加伤害、抓投利用障碍物、地面专用矢量加农炮，为不同高度赋予了战术意义。

## 纵方向的关卡设计
- **来源**: 《游戏设计的236个技巧》第3章/3.6节
- **核心**: 通过精心设计的跌落点和纵方向探索路径，将日常以平面为基准的移动转化为独特的纵方向冒险体验。
- **详述**: 《黑暗之魂》关卡设计中最具个性的是讲究纵方向。游戏比其他游戏更容易"跌落"，跌落点分为"立即死亡的跌落点"（煽动恐惧和紧张感）和"不会死亡的跌落点"（带来发现惊喜）。两种跌落点的鲜明对比创造了"纵方向探索的乐趣"。游戏中还通过设置窗户边可见但需绕路才能获得的道具、建筑物入口与墙壁风格一致需慢慢发现等方式，让玩家意识到纵方向观察力的重要性。开发者还加入"七色石"和下落攻击等机制鼓励纵向移动，同时不设跳跃功能以保持跌落的探索意义。
- **适用场景**: 3D探索型游戏、希望增加立体感的地下城设计。
- **示例**: 《黑暗之魂》城外不死镇中，战斗中跌落后发现新道路和道具，透过窗户看到道具后需从跌落点重新进入建筑物获取。

## 新人杀手陷阱的设计规则
- **来源**: 《游戏设计的236个技巧》第3章/3.6节
- **核心**: 陷阱必须遵守"可识破""有闪避机会""不会一击毙命""可反利用"的规则，让玩家理解并接受死亡原因。
- **详述**: 《黑暗之魂》中新手陷阱随处可见，但玩家仍想重新挑战。原因是所有陷阱遵守严格规则：仔细观察即可识破（如调整镜头可看到触发机关）；踩到陷阱后有闪避时间；即使被命中一次也不会死（玩家在该阶段生命值足够承受第一击）；应用得当可反过来消灭敌人。正因玩家能从游戏机制中理解并接受死亡原因，才会萌生"下次小心一点"的重新挑战念头。真正意义上的新人杀手（不可见、不可闪避、一击毙命）几乎没有玩家能接受。若想让玩家在失败后想重新挑战，需通过关卡设计将死因明确传达给玩家。
- **适用场景**: 高难度动作游戏中的陷阱设计、惩罚性关卡设计。
- **示例**: 《黑暗之魂》塞恩古城中箭矢陷阱：踩下开关后射出三支箭，第一支箭不会致死，闪避后还可反过来用陷阱消灭敌人。

## 全无缝关卡设计与超第一人称
- **来源**: 《游戏设计的236个技巧》第3章/3.6节
- **核心**: 所有地图全无缝衔接可让玩家始终沉浸于游戏世界，配合高自由度的通关路径为每名玩家打造专属冒险。
- **详述**: 《黑暗之魂》25张立体衔接的地图没有读取画面，让玩家意识持续保持在游戏之中。开阔地区能眺望远处实际可抵达的地图景色。玩家从游戏一开始就被扔入无缝世界，可获得多个无先后顺序的目标，从任何地图开始探索。由于通关路径高度自由，每名玩家的冒险都不同。这种让玩家充分体验亲自做出选择和决定的关卡设计称为"超第一人称关卡设计"——玩家角色名字、外观、能力都由玩家设置，是玩家的化身。要实现超第一人称，需给玩家提供"亲眼去看""亲自思考""亲自选择决定行动""自发采取行动"的自由度。
- **适用场景**: 高自由度探索型RPG、开放世界游戏、强调玩家自主性的游戏。
- **示例**: 《黑暗之魂》中每名玩家可选择不同路线通关，有人先去城外不死镇，有人误入小隆德遗迹遇到强敌。

## 恐怖循环
- **来源**: 《游戏设计的236个技巧》第3章/3.7节
- **核心**: 恐怖游戏关卡设计需建立"诱发不安→提供想象最糟糕事态的材料→恐怖事件→行动"的循环，并不断堵住退路以强化恐惧。
- **详述**: 恐怖电影和游戏中共通的恐怖循环：先让玩家处于不安状态（阴森环境、无BGM的死寂）；然后提供想象最糟糕事态的材料（看到丧尸啃食人头）；接着恐怖事件发生（丧尸袭击）；最后玩家的行动（逃跑或战斗）不但不能改善事态反而让情况恶化（逃进小黑屋），诱发更深层的恐惧。堵住退路是关键——救援直升机见死不救、同伴失踪、出口被封锁，只有被逼至走投无路时玩家才会真正产生恐惧感。在关卡设计中让恐怖循环不断重复并扩大，是让玩家感到恐惧的秘密。
- **适用场景**: 恐怖游戏的关卡设计、需要营造紧张氛围的叙事游戏。
- **示例**: 《生化危机4》村莊战斗：警车被推下谷底（堵退路）→看到被烧死的警官（想象最糟事态）→村民集团袭击（恐怖事件）→逃入民宅反而被包围（行动恶化事态）→钟声响起暂时缓解。

## 诱导玩家制定战术的关卡设计
- **来源**: 《游戏设计的236个技巧》第3章/3.7节
- **核心**: 关卡设计的"高自由度"即"有多种战术可用"，通过地形和障碍物引导玩家自然形成不同战斗方式。
- **详述**: 《生化危机4》仅初期村莊中央地图就诱导出5种战术：逃跑战术（远离敌人集团或逃入室内保持距离射击）、保持距离战术（借助篱笆等障碍物用动作键翻越制造距离）、减小受攻击面积战术（背靠墙壁将四方向威胁减为两方向）、固守阵地战术（在民宅内用家具堵门、从墙洞射击）、高低差战术（逃至二楼阳台踢倒梯子或将敌人踢下）。关卡设计将玩家逼入绝境，促使其亲自思考并制定战术。将玩家逼入绝境从而引出"本性"的关卡设计，让有些玩家冷静虐杀敌人，有些惊慌尖叫四处逃跑，如同丧尸电影中的桥段。
- **适用场景**: 射击游戏、TPS/FPS中需要丰富战术选择的战斗场景。
- **示例**: 《生化危机4》村莊中央地图中，篱笆、民宅、二楼阳台、窗户等关卡元素自然引出不同战术。

## 安心与恐惧的交替
- **来源**: 《游戏设计的236个技巧》第3章/3.7节
- **核心**: 有效制造恐惧需要在玩家感情中建立对比：让玩家经历安心、放松警戒后再发动突然袭击。
- **详述**: 简单的机制是让玩家连续体验安全从而放松警戒——如多次进入安全小屋后，最后一间才埋伏敌人。还可利用敌人"装死"：玩家经过时尸体不动（安心），返回时突然爬起来攻击（恐惧崩溃）。或利用迂回道路上的巡逻敌人从背后袭击。但人类无论面对何种恐惧都会习惯，《生化危机4》在玩家开始适应恐怖时，关卡设计会渐渐向射击乐趣方向转变。这种"看透玩家心境变化规律的关卡设计"在恐怖之余将动作的惊喜与兴奋带给玩家。
- **适用场景**: 恐怖游戏、需要情绪起伏的叙事驱动游戏。
- **示例**: 《生化危机4》中连续安全的小屋后最后一间埋伏敌人；"装死"的尸体在玩家返回时突然爬起攻击。

## TPS中"右侧"的关卡设计
- **来源**: 《游戏设计的236个技巧》第3章/3.7节
- **核心**: TPS游戏玩家角色占据屏幕左侧形成大范围死角，关卡设计需据此调整通道方向和敌人配置。
- **详述**: 绝大多数TPS中玩家角色显示在屏幕偏左侧，瞄准时前方出现死角。左出的通道中玩家既难看到左侧也难以攻击；掩体左侧来的敌人弹道被身体阻挡，可射击距离比右侧短很多。右转时看到的范围比左转大得多。据此，如果关卡设计中大部分通道为右出且敌人不从左侧进攻，则相对简单；如障碍物少且敌人可从左右进攻，则难度大增。还可通过改变敌人站位将简单地图变为困难地图，创造"过去容易回来难"的效果。
- **适用场景**: TPS射击游戏的关卡设计。
- **示例**: 《生化危机4》中根据TPS偏左画面特征设计关卡，右侧通道更友好，左侧通道更危险。

## 障碍物/掩体影响战术
- **来源**: 《游戏设计的236个技巧》第3章/3.8节
- **核心**: 在射击类游戏中关卡设计是玩家制定战术的大前提，障碍物/掩体的形状、高度、方向性、摆放方式大幅左右战术。
- **详述**: 没有障碍物的射击游戏只是单纯考验反应速度。加入掩体后玩家可制定"寻找敌人""隐藏自己""引诱敌人"等战术。掩体的关键属性包括：形状高度（立方体为主，高度决定可掩护/可攀爬）、方向性（掩体面与射击方向接近垂直时有效）、可贯穿/可破坏性（木篱笆可被打穿、可被破坏的掩体需精确计算使用时间）。摆放方式对战术影响最大：面对面平行摆放时优势相等，略微倾斜则每个掩体优势不同；多个障碍物不同角度摆放时朝向交叉点易成火力集中处。利用掩体摆放可将防卫据点设计成对攻或对守方有利的形式。
- **适用场景**: TPS/FPS射击游戏的战斗场景设计。
- **示例**: 《神秘海域：德雷克的欺骗》中掩体的不同摆放方式创造了平等对战、扇形防卫、通道碾压等多种战斗形式。

## 故事曲线与调整步调
- **来源**: 《游戏设计的236个技巧》第3章/3.8节
- **核心**: 游戏关卡设计需像电影一样管理故事曲线，用难关、节奏变化和反差创造戏剧性，并让游戏过程的基调与剧情基调吻合。
- **详述**: 剧情由幕、段落、场景、分镜构成，场景和分镜营造的节拍是关键。创造剧情节拍的工序就是"调整步调"。游戏中玩家的兴奋程度往往在战斗结束或获得成就后下降，所以需在游戏过程末尾添加"调节"——如高潮动作直接连接过场动画，让玩家在生死未卜的状态下进入剧情，保持激动情绪。难关需要节奏变化（不能等间隔出现）、大小区分（强敌vs剧情事件）和反差（明暗对比、敌友对立）。游戏基调要尽量与故事基调吻合——如果主人公刚失去挚爱，接下来的游戏过程应是激烈战斗而非轻松跳跃。将"故事的热度"与"玩家所处高度"及"运动矢量"相关联也是一种有效手法。
- **适用场景**: 叙事驱动的动作游戏、电影化体验的游戏设计。
- **示例**: 《神秘海域：德雷克的欺骗》第六章中，从燃烧崩塌的古城房顶飞奔后纵身一跃直接连接过场动画，保持玩家在生死未卜的紧张感中进入剧情对话。

## 谜题设计的可达性原则
- **来源**: 《游戏设计艺术》第14章/透镜54
- **核心**: 谜题或关卡的"可达性"是指玩家能否清楚地看到最初几步该做什么——好的可达性让玩家知道从何入手，而不会对着谜题发呆。
- **详述**: 可达性包含三个层次：玩家是否知道该做什么（目标是否清晰）、是否知道从哪里开始（第一步是否显而易见）、是否被吸引去操作它（谜题本身是否像个好玩具）。《十五数字推盘》虽然解题不简单，但如何上手非常清晰。相比之下，许多文字密码谜题目标明确但玩家完全不知从何开始。Scott Kim的名言："要设计一个好谜题，首先要打造一个好玩具"——好的谜题/关卡应该让玩家一看到就被吸引去操作，而操作方式本身就很直观。
- **适用场景**: 设计关卡开头、谜题呈现方式、新手引导区域时。
- **示例**: 魔方之所以成功，部分原因在于可达性——即使不会解的人也会被吸引去旋转它，而旋转操作本身直观明了。

## 可见进步与平行性缓解挫折
- **来源**: 《游戏设计艺术》第14章/透镜55、透镜56
- **核心**: 玩家在解谜或闯关时需要看到自己在进步，而当一条路走不通时，平行提供的多条相关路径让玩家可以换路而非卡死。
- **详述**: 可见进步是谜题和关卡设计中最关键的体验之一。研究表明可见的进步（或缺失它）是影响心情的最大因素。魔方的成功在于玩家可以看到越来越多的颜色归位。平行性是另一种关键设计——同时给玩家多个相关挑战，当一条路走不通时可以换另一条。"改换工作益如休养"在此完美适用。RPG游戏很少只给玩家一个挑战，更常见的是同时给出两个或更多平行任务。平行的挑战之间还可以互相联系——战胜一个挑战取得的进步能让解决另一个挑战更容易。
- **适用场景**: 设计关卡分支结构、任务系统、防止玩家卡关的保障机制时。
- **示例**: 纵横字谜和数独天然具有平行性——卡在一个问题时可以先做其他的，往往在解题过程中会获得其他问题的线索。

## 金字塔谜题结构——从小挑战到大高潮
- **来源**: 《游戏设计艺术》第14章/透镜57
- **核心**: 金字塔结构让一系列小谜题的解法线索汇聚成一个更大的终极挑战，把短期目标和长期目标巧妙地统一起来，大幅延长玩家的兴趣。
- **详述**: 金字塔谜题的核心特征是：底部的多个小挑战各自给出部分线索，这些线索汇聚到顶点构成最终挑战。报纸上的Jumble游戏就是经典案例——先解开四个单词，每个单词中圈出的字母再组成最终短语。这种结构将短期目标（解开每个小谜题的成就感）和长期目标（最终答案的好奇心）结合，逐步增加难度，并且有一个单一明显且有意义的顶点。大型关卡可以使用嵌套的金字塔——小金字塔的顶点成为更大金字塔的基石。
- **适用场景**: 设计关卡的整体任务结构、Dungeon（地下城）的Boss解锁流程时。
- **示例**: 《塞尔达传说》系列的地下城设计是金字塔结构的典范——解开多个小谜题获得钥匙/道具，最终用所有收集的要素挑战Boss。

## 机关旋律——彼得与狼式关卡编排
- **来源**: 《通关！游戏设计之道》第12关
- **核心**: 像编曲一样编排关卡元素，让玩家体验从简单到复杂的自然递进，最终达到"心流"状态。
- **详述**: 斯科特·罗杰斯借用普罗柯菲耶夫《彼得与狼》的配乐方法来组织关卡：每种游戏元素对应一种"乐器"，从简单开始逐步叠加。具体步骤：1) 玩家刚进入时只给简单挑战（走、跳、收集物品）；2) 加入一个机关，重复几次让玩家学会运作方式；3) 加入第二个机关并让玩家学会，然后将两个机关结合；4) 加一个陷阱增加刺激，让玩家学会躲避；5) 引入敌人，让玩家学习战斗；6) 将敌人和陷阱结合，让一切更刺激；7) 当玩家熟悉所有元素后，随机组合保持警惕。只需几种类型的机关和敌人就能制造出丰富体验，关键在于如何组合。
- **适用场景**: 关卡元素的引入顺序设计，尤其适合线性关卡的节奏编排。
- **示例**: 《古惑狼》《神秘海域》中用喷火坑、移动障碍、旋转刀片三种机关的不同组合创造出数十种不同体验；Treasure工作室的游戏（《火枪英雄》《炸弹小子》）展示了如何高效利用有限机关。

## 陷阱视觉设计第一法则
- **来源**: 《通关！游戏设计之道》第12关
- **核心**: 陷阱必须"看上去就很危险"——利用形状、颜色、音效和粒子效果，让玩家一眼识别危险。
- **详述**: 陷阱设计的第一法则是视觉可读性：带尖儿的、带刺儿的、喷火的、结霜的、闪着火星儿的、冒着毒气的——如有必要直接贴个骷髅头。创造陷阱时应从关卡主题中寻找灵感，使用看起来属于该关卡环境的东西。同时也要从现实中寻找危险事物的灵感（多刺的仙人掌、带尖的铁丝网）。反面教材：有设计师将陷阱做成黄黑相间条纹（因为"蜜蜂就是这样的"），但玩家并不会联想到危险——最终陷阱还是改成了代表危险的红色。一致性很重要：如果游戏中红色代表危险，所有危险事物就应该是红色。
- **适用场景**: 设计任何需要玩家直觉识别的危险元素——陷阱、敌人攻击、危险区域。
- **示例**: 《魔界英雄记》的地下城关卡中，陷阱设计与主题一致：从大门伸出的乱抓的手、挥舞砍手的装甲人、滴着毒液的带刺铁球、漂着棺材盖的毒水沟。

## 检查点设计的经验法则
- **来源**: 《通关！游戏设计之道》第12关
- **核心**: 检查点的放置必须让玩家感到安全和被尊重，而不是被惩罚。
- **详述**: 检查点分为可见和不可见两种。可见检查点让玩家产生安全感，但显得"刻意"；不可见检查点不打断沉浸感，但玩家不清楚会从哪里重生。无论哪种，都需遵循：检查点始终朝向玩家进来的方向（避免转身调整视角）；永远不要放在陷阱旁边或敌人警戒范围内（一重生就受伤太"流氓"）；玩家应直接在地面上重生（避免落地等待和冗长动画）；放在平台上避免碰撞问题；激活检查点时自动保存游戏数据（不让玩家额外操作保存界面）。
- **适用场景**: 所有需要重生机制的关卡设计。
- **示例**: 早期《生化危机》中开门作为关卡加载手段，同时营造紧张气氛——将加载过程巧妙融入玩法体验。

## 谜题设计核心原则——让玩家看见门再找钥匙
- **来源**: 《通关！游戏设计之道》第12关
- **核心**: 先让玩家看到目标（门），再把他们送到有解决工具的地方（钥匙），保证谜题要素简单独立、所有工具近在咫尺。
- **详述**: 谜题设计应遵循：让谜题要素尽量简单且各自独立，组合少数要素就能制造出不同谜题；一旦要素被调整，清楚反馈给玩家并让其知道如何恢复原状；所有要素保持一致标记，除非故意增加难度；谜题要素间的关系必须清楚，避免使用文化含义过强或晦涩的东西；解决谜题的所有工具必须近在咫尺，不要让玩家满世界跑。最重要的是给玩家"啊哈！"时刻——意识到各部分如何组合并发现解决方案。如果玩家找不到答案，不能因此卡住他们，可以给提示或直接告诉答案，但需付出代价（如额外奖励或现金）。
- **适用场景**: 设计任何游戏内谜题，从简单机关谜题到复杂解谜系统。
- **示例**: 《生化危机2》中警察局用国际象棋棋子开门的谜题是反面教材——警察局里根本不该有国际象棋；更好的设计是用死掉警官的ID卡开门，让谜题融入故事情境。

## 时机谜题四原则
- **来源**: 《通关！游戏设计之道》第12关
- **核心**: 移动机关/时机谜题必须有可预判的路径、可预判的时机、合理的窗口期，并通过环境线索给玩家安全感提示。
- **详述**: 时机谜题（旋转刀刃、坍塌桥梁等移动机关）是制造紧张氛围的利器，但必须遵循四原则：1) 固定移动路径——前后、上下、之字形或8字形，让玩家掌握规律；2) 可预判的通过时机——随机的陷阱对玩家不公平；3) 合理的窗口期——不能太长也不能太短，开始和结束各留回旋余地；4) 环境线索引导——血迹、地面沟槽、光照投影、音效、粒子效果、空间结构和装饰物，帮助玩家识别安全位置和危险区域。
- **适用场景**: 需要制造紧张感和节奏变化的关卡段落。
- **示例**: 《龙六历险记》中幽灵密室旋转桨叶关卡，是游戏史上第一个时机迷宫关卡，至今仍令玩家紧张。

## 机制与空间环境的有机匹配
- **来源**: 《游戏感》第15章
- **核心**: 关卡中物件的尺寸、间隔和特征应基于角色的运动能力精确调整——机制调整和关卡布局是密切关联的，应该同步迭代。
- **详述**: 《超级马里奥64》最初的原型是一个"玩法集中的乐园"测试关卡，里面包含了一个近乎成品的马里奥版本和大量物件。当跳跃高度和轨迹调整后，物件间的距离也同步调整；蹭墙跳和墙与墙之间的间隔是同时建立的。物件在空间上的基础关系构成了三种最主要的空间关系：垂直高度（基础跳跃高度 vs 后空翻/三段跳高度）、水平距离（基础跳跃距离 vs 远跳距离）、平台大小（安全着陆面积）。整个游戏中所有关卡都严格遵循这套空间指南——玩家能快速习惯各种跳跃的可预测高度，因为环境本身就是为这些跳跃量身定做的。早期关卡还用基础跳跃作为正常路径，但留下侧空翻或后空翻的捷径——用关卡设计奖励技巧更高的玩家。
- **适用场景**: 3D平台跳跃、动作游戏的关卡设计——先确定角色运动参数，再据此建立空间摆放规则。
- **示例**: 《超级马里奥64》中炸弹王国的正常路径用基础跳跃，但有一条用侧空翻更快上去的捷径；所有砖块高度都刚好合乎基础跳跃高度。

## 坡度作为软性边界——用物理法则替代硬性限制
- **来源**: 《游戏感》第15章
- **核心**: 用地形坡度和滑动规则作为关卡边界的软性限制，而非透明碰撞墙等不自然的硬性限制——玩家能通过物理逻辑理解"不能通过"，而非感到设计师的直接干预。
- **详述**: 《超级马里奥64》中角色有摩擦系数，当斜坡超过约75度时，向前推力不足以克服反向力，角色会进入下滑状态。这种坡度被用作关卡边缘的软性边界：你不能翻过炸弹王国的墙，因为上去后会往回滑。试一次就知道徒劳无功，玩家能快速通过符合逻辑的物理关系理解限制，而非通过痛苦的人为约束（如空气墙或隐形碰撞体）。坡度还用作软性惩罚——下坡过快会导致失控滑落，提供了风险回报：冒险全速下坡可以更快，但可能滑出边缘。这种设计的精妙之处在于把限制融入了物理法则中，保持了玩家对游戏世界统一性的信念。
- **适用场景**: 设计关卡的边界、区域限制和引导玩家路径时，优先考虑物理性的软限制。
- **示例**: 《超级马里奥64》中无法爬上的陡坡自然限制玩家活动范围；雪山关的企鹅赛跑充分利用了下坡滑动的风险回报。

## 地标中心与高处鸟瞰的关卡布局
- **来源**: 《游戏感》第15章
- **核心**: 以高大显眼的地标为中心设计关卡，并提供高处鸟瞰点——这不仅改善摄像机表现，还赋予玩家强烈的高层次空间感和探索吸引力。
- **详述**: 《超级马里奥64》中大多数关卡都以高大圆滑的构造为中心——要塞里的高塔、炸弹王国中间的塔顶、雪山的巨大雪人。站在这些高点上能看到关卡里所有可获取星星的位置，建立起强烈的空间认知。这种地标中心设计有两个好处：一是高大圆滑的结构避免摄像机被障碍物阻挡，大幅改善3D摄像机表现；二是给予玩家类似"站在山顶俯瞰"的强大空间感受。当角色身处巨大建筑顶部时，摄像机能自由俯视，让下方景物如同蚂蚁般渺小。这种设计让关卡像"主题公园的区域"——有一个显眼的中心地标，周围环绕着不同的挑战和探索点。
- **适用场景**: 3D游戏的关卡整体布局设计，尤其适合开放或有探索要素的3D场景。
- **示例**: 《超级马里奥64》每个主要关卡都有一个中心高塔，站在上面能俯瞰全关——要塞的高塔、炸弹王国的中间塔顶都是典型的地标中心设计。
