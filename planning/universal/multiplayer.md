# 多人游戏设计

> 来源：《体验引擎》第7章

## 博弈论基础
- **来源**: 《体验引擎》第7章
- **核心**: 多人游戏的核心在于玩家之间对彼此行为的预测和回应，而非单纯的机制操作
- **详述**: 博弈论（Game Theory）是分析多人游戏中行动与对策交互行为的数学门类。当玩家面对的不是固定机制的系统（如摧毁无人的城堡），而是会主动预判和回应的高智商对手时，博弈论便派上了用场。博弈论者用收益矩阵来描述这种"策略交互"——博弈论意义上的"游戏"并非我们设计的整款游戏，而是游戏中一个特定的决策点。例如足球赛中罚点球时射门方向与扑救方向的选择，就是一次独立的策略交互。
- **适用场景**: 分析多人游戏中任何需要玩家预测和回应对手行为的决策点
- **示例**: 围城战中，攻方选择用攻城锤还是工兵偷袭，守方选择防守大门还是部署弓箭手——双方都不确定对方的选择，却必须提前做出决策

## 纳什均衡
- **来源**: 《体验引擎》第7章
- **核心**: 如果在某策略配置中，任何玩家单独改变策略都无法获得更高收益，则该配置为纳什均衡——游戏玩法总会向此状态收敛
- **详述**: 纳什均衡是稳定且自我巩固的，因为所有玩家都没有动机改变。非纳什均衡的配置则不稳定，玩家总会找到改变的理由。因此，一个游戏中的策略交互要么需要具备多种纳什均衡，要么应彻底清除纳什均衡。只含有一种纯粹纳什均衡的交互是设计败笔，因为每个玩家都清楚该怎么做，决策失去了意义。
- **适用场景**: 审查游戏中的策略交互是否存在唯一的纳什均衡，若存在则需重新设计或调整平衡性
- **示例**: 两个猎人选择捕鹿或捕兔子。两人都捕兔子虽非最优结果（捕鹿收益更高），但也是纳什均衡——因为若只有一方改变去捕鹿，他反而会挨饿

## 石头剪子布机制
- **来源**: 《体验引擎》第7章
- **核心**: 通过相互克制的功能三角消除纳什均衡，使每个玩家总有机会通过预测对手获得回报
- **详述**: "石头剪子布"是对称游戏中最简单的消除纳什均衡的结构。格斗游戏中防御克制出拳、摔投克制防御、出拳克制摔投；战略游戏中枪兵克制骑兵、弓箭手克制枪兵、骑兵克制弓箭手——这类三角克制结构出现在无数游戏中。对于非对称游戏，则需要"猜硬币"模式：一方希望匹配（如防守方希望猜对进攻点），另一方希望错开（如进攻方希望从无防备处进攻）。这两种模式（石头剪子布和猜硬币）是策略交互中仅有的优雅设计模式，增加更多元素并不会丰富预测性和谋略性。
- **适用场景**: 设计对称或非对称的对抗性策略交互时，作为消除纳什均衡的基础框架
- **示例**: 《星际争霸2》中虫族vs人族：异龙克制攻城坦克、陆战队员克制异龙、爆虫克制陆战队员、攻城坦克克制爆虫——四种单位的反复交互可产生数百小时的策略变化

## 混合策略
- **来源**: 《体验引擎》第7章
- **核心**: 当不同策略的收益不等时，最优解不是固定选择最高收益策略，而是按特定概率随机选择多种策略
- **详述**: "石头剪子布"虽无纯纳什均衡，却存在混合纳什均衡：以各33.3%的概率随机出招。当不同结果对应不同收益时（如出石头赢了得5块，出剪子或布赢了得1块），天真地总出高收益策略会被对手利用。正确做法是按特定概率混合各种策略，使得每一种可选出招在均衡状态下具有相同收益。人类虽无法精确计算，但凭直觉常能接近最优混合策略。
- **适用场景**: 设计不对称收益的策略交互，或理解为什么玩家不应该总是选择"最优"策略
- **示例**: 足球罚点球时，球员应将球踢向擅长方向约60%的时间、不擅长方向约40%——这样无论守门员扑向哪侧，成功率都稳定在约74%。如果偏离此比例，守门员便可利用

## 心理战
- **来源**: 《体验引擎》第7章
- **核心**: 心理战是在博弈论数学分析之上，通过预测对手思维、欺骗对手以及使用计谋来获取额外优势的思维博弈
- **详述**: 博弈论交互只是框架，游戏的情感价值通过框架之上的心理战得以体现。心理战之所以能够起作用，是因为真实游戏比数学模型模糊得多——结果不可精准量化，策略之间没有清晰分类，信息也不充分。心理战的产生条件包括：策略能够流畅组合（如《星际争霸2》中单位数量的微调创造出无限策略空间）、收益复杂且难以量化（位置、数量、队形、经济等因素交织）、人类无法制造真正的随机数（可被利用的概率偏差），以及信息可被操纵（获取信息、拒绝给予信息、散布假信息）。
- **适用场景**: 在核心博弈论交互之上构建能够引发深度思维博弈的模糊边界，提升技巧上限
- **示例**: 《现代战争2》中两名玩家在一个两入口小屋旁的遭遇——表面是射击游戏，实质是关于留守还是冲锋、防守哪个入口、是否绕后、是否逃跑的多层心理博弈，高水平玩家会考虑到所有这些可能性

## 信息操纵
- **来源**: 《体验引擎》第7章
- **核心**: 玩家操纵信息的三种基础方式——获取信息、拒绝给予信息、散布假信息——是心理战的核心工具
- **详述**: 获取信息（派出侦察兵、聆听脚步声、使用心跳传感器）提升自身决策效果；拒绝给予信息（杀死对方侦察兵、投掷烟雾弹、比赛前期保留实力）使对手难以决策；散布假信息（射击游戏中打开一扇门制造错觉、策略游戏中故意建造诱饵机场）则直接控制对手的判断。多层次蒙骗甚至可以让对手认为你掌握了某些信息而实际并不了解，或让他以为你没注意到某些信息而你已洞察一切。
- **适用场景**: 设计支持多层次信息博弈的游戏机制
- **示例**: 《星际争霸》职业选手在比赛前期保留实力不展示全部战术，直到决赛才全力以赴——这是在拒绝给予信息

## 破坏性行为
- **来源**: 《体验引擎》第7章
- **核心**: 多人游戏中玩家的自我创造目标会打破游戏平衡，必须通过设计手段而非仅靠社交约束来应对
- **详述**: 多人游戏结构紧凑且平衡性好，一旦有玩家追求计划外的目标，平衡就会被打破并影响所有人。破坏性行为分为两类：目标分歧（玩家追求与游戏既定目标不一致的目标）和技巧差异（水平悬殊的玩家被迫同场竞技）。防御蓄意破坏的第一条准则是鼓励玩家实现有意义的目标——如果按既定目标玩就是达成征服、探索、交流的最佳途径，玩家自然会遵从。若无法完美统一目标，则需增加玩家数量以稀释单个异常玩家的影响，或直接在机制上禁止破坏行为（如允许绕过被堵的门）。最终防线是投票踢人等治安系统。
- **适用场景**: 任何涉及多人交互的游戏，特别是匿名匹配的在线游戏
- **示例**: 《魔兽世界》中玩家对从高处落下的同伴使用"水上行走"技能，导致同伴像摔在钢板上一样跌落水中——这是典型的蓄意破坏（Griefing）

---

## 目标分歧
- **来源**: 《体验引擎》第7章
- **核心**: 当某些玩家追求的目标与游戏既定目标不一致且破坏他人体验时，目标分歧就会出现
- **详述**: 目标分歧的程度从轻微到严重不等。轻微的分歧如《使命召唤》中有人专注于杀敌数而非夺取旗帜——虽然偏离目标，但杀敌通常有助于团队获胜，影响有限。严重的分歧如《求生之路》中某个玩家偷偷跑掉单独行动——这直接瓦解了"四人协力求生"的核心设计。最恶劣的是蓄意破坏（Griefing）：为了自身乐趣而故意破坏他人体验，如堵门、攻击队友、在人流密集处放置惊悚图片等。面对面的游戏有社交约束力，但网络匿名环境下必须依靠游戏设计来防御。
- **适用场景**: 评估多人游戏是否存在激励偏离既定目标的机制，以及是否存在可被利用来伤害他人的系统
- **示例**: 《求生之路》通过AI接管离开玩家的角色来缓解因玩家退出导致的目标分歧——虽然AI不如真人，但作为权宜之计可接受

## 技巧差异
- **来源**: 《体验引擎》第7章
- **核心**: 水平悬殊的玩家被迫同场竞技时，新手和老手的目标分歧会同时损害双方的体验
- **详述**: 低水平玩家的目标是学习和了解游戏、不被压力压垮；高水平玩家则追求有深度和技巧延伸性的对局。这两种目标直接冲突：新手遭受侮辱和压力，老手因对手太弱感到无聊或因队友太差感到沮丧。缓解技巧差异的方法包括：简洁优雅的设计（降低入门门槛）、匹配算法（让水平接近的玩家对战）、结构设计（新手区先行学习再组队），以及适应性训练（如《求生之路》在关键时刻显示操作提示）。若无法缩小差距，可降低玩家间相互依赖——最好的设计是"分享胜利，但不分担失败"。
- **适用场景**: 设计匹配系统、新手引导、团队结构时，评估不同水平玩家共存的影响
- **示例**: 《光环：致远星》的组队战斗模式中，玩家几乎可以忽略队友，但两三人可随时临时组队合力击败敌人后分道扬镳——非强制的短期合作分享了胜利喜悦而无需承担失败痛苦

## 竞争、合作与竞争合作的三角平衡
- **来源**: 《游戏设计艺术》第13章/透镜43、44、45
- **核心**: 游戏可以基于竞争（玩家对抗）、合作（玩家协作）或竞争合作（团队间竞争）设计多人互动——最佳多人游戏同时利用这三种力量的组合。
- **详述**: 竞争激发玩家提升自我——当对手水平接近时竞争最激烈，水平悬殊则双方都不满意。合作创造社交纽带——当玩家必须真正依赖彼此时合作最有意义（而非可有可无）。竞争合作结合了两者——团队内部合作，团队之间竞争，这是最自然的多人游戏形式，因为玩家既有队友支持又面临对手挑战。设计多人模式时应考虑：玩家是否有足够的竞争理由？合作是否真正有意义（缺少任何一人都会失败）？竞争与合作之间的平衡是否让大多数玩家感到有趣？
- **适用场景**: 设计多人游戏模式、团队机制、合作关卡时。
- **示例**: 《求生之路》强制四人合作（任何一人倒下都可能全灭）同时又面临感染者团队的竞争压力，是竞争合作的经典案例；《马里奥赛车》以竞争为主但道具系统创造了间接的合作（第二名集火第一名）。

## 奖励比惩罚更有效
- **来源**: 《游戏设计艺术》第13章/透镜46、47
- **核心**: 心理学研究一致证明奖励比惩罚更能推动行为——能用奖励替代惩罚时就替代，当惩罚不可避免时要确保公平、可理解、可防止。
- **详述**: 暴雪在《暗黑破坏神》中发现：饥饿惩罚系统（不吃食物就能力下降）让玩家觉得烦人——必须做枯燥活动否则遭受惩罚。改为食物提供临时能力提升后，同样吃东西的行为从消极变为积极。奖励的类型包括：赞美、分数/点数、延长游戏时间、新能力/新区域权限、视觉/听觉演出。变化奖励比固定奖励更有力量（随机1/3概率获得30分比每次10分更吸引）。但当必须有惩罚时，所有惩罚都应该是玩家能理解并可以防止的——如果玩家感觉惩罚是随机的且无法阻止，会立刻给游戏贴上"不公平"的标签并离开。
- **适用场景**: 设计多人游戏的奖惩系统、排名机制、违规处理时。
- **示例**: 《卡通城》在线版将"死亡"设计为轻微惩罚的组合——传送回操场（小倒退）+道具消失（小损失）+角色垂头（羞辱）+暂时走得很慢（暂时失去能力），这种轻度组合刚好鼓励谨慎但不至于过度谨慎。
